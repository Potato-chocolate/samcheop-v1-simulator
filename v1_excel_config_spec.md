## 1. 문서 목적

본 문서는 v1 삼첩분식 Excel 자료에서 확인한 **시트 구조, 핵심 수치, 공식, 단위**를 웹 시뮬레이터 구현 기준으로 정리한 문서입니다. 또한 현재 React/Tailwind 프론트엔드에 반영된 계산식을 **매출 시뮬레이션 CONFIG**와 **개설비용 시뮬레이션 CONFIG**로 나누어 설명하고, 상담 현장에서 사용자 확인이 필요한 가정과 누락 데이터를 명확히 분리합니다.

> 본 기준서는 상담용 추정 도구의 내부 구현 근거입니다. 실제 가맹 계약, 견적, 수익 보장을 의미하지 않으며, 최종 견적은 본사 정책, 상권, 임대 조건, 공사 범위, 배달권역 정책에 따라 별도 확인되어야 합니다.

## 3. 매출 시뮬레이션 CONFIG

## 7. 사용자 확인이 필요한 가정과 질문

현재 웹 구현은 v1 Excel 자료를 상담용 웹 도구로 옮기기 위한 1차 기준을 적용했습니다. 다만 실제 상담 정확도를 높이려면 다음 항목은 사용자 또는 본사 담당자의 확인이 필요합니다.

| 확인 영역 | 현재 가정 | 확인 질문 | 영향 범위 |
