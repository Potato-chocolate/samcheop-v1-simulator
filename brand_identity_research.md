# 삼첩분식 브랜드 아이덴티티 조사 노트

## 핵심 브랜드 정의

삼첩분식은 공식 브랜드 페이지에서 스스로를 **“떡볶이의 새로운 기준”**으로 정의한다. 브랜드 메시지는 기존의 매운맛 중심 떡볶이 시장에 **맛의 다양성**, **취향별 선택**, **색다른 경험**을 제공하는 데 집중되어 있다.[^1] 앱 소개 문구 역시 “ONE, TWO, THREE! SAMCHEOP!”과 “떡볶이의 새로운 기준”을 반복하며, 삼첩이라는 숫자형 구조와 경쾌한 주문 경험을 브랜드의 기억 장치로 사용한다.[^2]

| 항목 | 조사 결과 | v1 시뮬레이터 반영 방향 |
|---|---|---|
| 브랜드 슬로건 | 떡볶이의 새로운 기준 | 단순 계산기가 아니라 “창업 판단의 새로운 기준”처럼 명확한 판단 도구로 설계 |
| 핵심 경험 | spicy, sweet, samcheop / 취향별 선택 | 입력값을 조합하는 경험을 “1첩·2첩·3첩” 단계형 구조로 표현 |
| 브랜드 자산 | 레드 메인 컬러, 옐로우 포인트, 삼색 심볼 | 컬러 체계는 떡볶이 레드·치즈 옐로우·패키지 크림/종이 톤 중심 |
| 차별 요소 | 프리미엄 종이 패키지, 시즌 박스, 협업 이벤트 | 박스·스티커·영수증·메뉴 카드 같은 패키지 모티프를 UI 패턴으로 사용 |
| 창업 메시지 | 안정적·체계적 본사, 전국 1% 본사, 장사 집중 | 결과 화면은 화려함보다 신뢰 가능한 손익표·현금흐름표 중심 |

## 디자인 인사이트

공식 페이지는 삼첩분식의 메인 컬러가 **떡볶이를 상징하는 레드**이며, **옐로우 포인트**로 경쾌하고 밝은 무드를 만든다고 설명한다.[^1] 따라서 v1 프론트엔드는 단막 v2의 디자인을 그대로 옮기지 않고, 삼첩분식 고유의 **레드·옐로우·종이 패키지·스티커 콜라주**를 바탕으로 다시 설계해야 한다.

브런치 분석 글은 삼첩분식의 성장을 **프리미엄 패키지**, **합리적 프리미엄**, **토핑과 아이디어**, **펀슈머 트렌드**로 설명한다.[^3] 이 관점은 시뮬레이터 화면에도 중요하다. 예비 점주는 숫자를 보되, 동시에 “이 브랜드가 왜 객단가와 재구매를 만들 수 있는가”를 직관적으로 느껴야 하므로 UI는 단순 금융 계산표가 아니라 **브랜드 패키지형 투자 판단 툴**이어야 한다.

## 창업/매출 관련 공개 참고값

마이프차에 따르면 삼첩분식은 월평균 매출 1,994만원, 매장 238개로 소개되어 있으며, 15평 기준 예상 창업 비용은 1,100만원~2,200만원, 수익 예시는 기준 매장 월매출 2,100만원·월수익 550만원, 고매출 매장 월매출 4,650만원·월수익 1,152만원으로 제시되어 있다.[^4] 다만 실제 v1 시뮬레이터의 수치는 사용자가 제공한 v1(삼첩분식) 폴더 자료를 우선 기준으로 삼아야 한다.

| 공개 참고 항목 | 값 | 사용 방식 |
|---|---:|---|
| 월평균 매출 | 1,994만원 | 외부 벤치마크 또는 기본값 검증용 |
| 기준 매장 월매출 | 2,100만원 | 시뮬레이션 초기 프리셋 후보 |
| 고매출 매장 월매출 | 4,650만원 | 낙관 시나리오 프리셋 후보 |
| 원재료비율 | 40% | v1 자료와 충돌 시 v1 자료 우선, 외부 검증값으로 활용 |
| 기타비율 | 20% | v1 자료와 충돌 시 v1 자료 우선 |
| 임대료 예시 | 80만~150만원 | 상권/면적별 입력값 기본 범위 후보 |

## v1 디자인 방향 재정의

초기에는 단막 v2 온보딩 관점에서 한국형 네오 브루탈리즘 대시보드를 고려했으나, 삼첩분식에는 보다 명확히 **“프리미엄 분식 패키지 스튜디오”** 방향이 맞다. 화면은 빨간 떡볶이 소스의 에너지, 노란 치즈/튀김의 포인트, 종이 박스의 따뜻한 질감, 시즌 콜라보의 스티커 감각을 결합한다. 단, 창업 상담용 도구인 만큼 과도한 캐릭터성보다는 **투자 판단의 신뢰감**과 **브랜드 팬덤의 경쾌함** 사이의 균형을 유지해야 한다.
