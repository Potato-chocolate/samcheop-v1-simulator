# 배포 환경 프론트엔드 QA 확인 기록

| 구분 | 확인 위치 | 수정 방향 |
|---|---|---|
| 내부 버전 문구 | 상단 로고 `삼첩분식 v1`, 히어로 배지 `삼첩분식 창업 상담용 v1`, 본문 `v1 삼첩분식 엑셀 자료`, 매출 섹션 설명 `v1 목표매출`, 자료 기준 `v1 엑셀 자료`, 하단 푸터 `v1`, `v2 단막` | 가맹상담용 표현으로 치환하고 v1/v2/단막 표현을 화면에서 제거 |
| PDF 버튼 | 상담 리포트 저장 카드의 `PDF 저장/인쇄` 버튼 | 버튼 레이블을 `PDF 저장`으로 단순화 |
| 로그인·클라우드 저장 | `저장 리포트`, `리포트 저장·링크 복사`, `최근 저장 링크 복사`, 공유 링크 생성 및 로그인 유도 | 상담 현장 흐름을 위해 인증·클라우드 저장·링크 공유 UI 제거 |
| 숫자 포맷 | 입력 필드 값이 `32344100`, `20000`, `500000`처럼 쉼표 없이 표시 | 입력 필드와 계산 결과에 천 단위 구분이 보이도록 표시 로직 개선 |
| 한글 줄바꿈·가독성 | 히어로 대형 제목과 카드 제목의 단어 중간 줄바꿈 가능성, 지나치게 두꺼운 제목 가중치 | `word-break: keep-all`, `overflow-wrap: break-word`, 폰트 굵기 조정 적용 |
| 벤치마크 | 수도권 상위 매장 벤치마크가 5위까지만 표시 | 10위까지 표시하도록 데이터와 UI 확장 |
