# 삼첩분식 v1 시뮬레이터 구현 노트

## 구현 완료 범위

삼첩분식 v1 시뮬레이터는 v2(단막) 구조를 참고하되, 프론트엔드와 상담 흐름을 삼첩분식 브랜드에 맞게 재설계했다. 화면은 **1첩 입력값**, **2첩 손익 리포트**, **3첩 개설비용 리포트**로 구성했으며, 삼첩분식의 레드·옐로우 패키지 감각과 영수증형 상담 리포트 구조를 적용했다.

| 영역 | 구현 내용 | 기준 자료 |
|---|---|---|
| 매출 프리셋 | 중앙값, 평균, 상위 25%, 목표값, 상위 10% | v1 수도권 55개 매장 분석 |
| 손익 계산 | 식자재, 포장재, 배달앱 수수료, 인건비, 고정비, BEP | v1 목표 요약표 및 매출 산정자료 |
| 채널 믹스 | 배달·포장·홀 매출 비중 표시 | v1 매출 산정자료 |
| 개설비용 | 평수, 간판 형태, 홀 테이블 수, 본사비용 면제 여부 | v1 개설비용 엑셀 |
| 디자인 | 삼첩분식 브랜드 조사 기반 프리미엄 분식 패키지 콜라주 | 공식 웹/브랜드 시각 자료 조사 |

## 현재 적용 가정

매출 손익에서는 월매출 3,234만 4,100원을 기본값으로 두고, 객단가 2만원, 임대료 50만원, 공과금 70만 5천원, 정직원 0명, 파트타이머 2.5명을 기본 조건으로 설정했다. 식자재율은 36%, 포장재율은 4%이며, 장부·소모품 40만원, 푸드테크 2만 2천원, 로열티 11만원을 고정비에 포함했다.

개설비용에서는 12평, 플렉스 간판, 홀 겸업형, 2인 테이블 6개를 기본값으로 설정했다. 본사비용은 가맹비, 교육비, 감리비, 물류보증금을 정상가와 프로모션 차감액으로 분리해 보여준다. 실납부 예상 합계는 VAT 별도 기준으로 표시하고, VAT 10%는 안내용으로 계산한다.
