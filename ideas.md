**Interaction Philosophy**: 입력값을 바꿀 때마다 카드는 조용히 재정렬되고, 주요 수익 지표는 영수증 합계가 다시 찍히는 듯 갱신된다. 사용자가 과감한 매출 또는 비용 가정을 넣으면 스티커형 경고가 붙어 실무적 주의를 준다.

**Design Movement**: 레트로 분식 금융 터미널. 1990년대 분식집 메뉴판과 현대 핀테크 계산기의 대비를 결합해, 삼첩분식의 친근한 먹거리 감각과 창업 수익 분석의 전문성을 동시에 표현한다.

**Core Principles**: 첫째, 메뉴판처럼 빠르게 읽히되 계산기는 충분히 정밀해야 한다. 둘째, 가격표·옵션표·마진표를 하나의 운영판처럼 연결한다. 셋째, 레트로 무드는 장식으로만 쓰지 않고 메뉴 구조와 비교표에 반영한다. 넷째, 사용자가 복잡한 수치를 다룰 때도 화면의 길을 잃지 않도록 좌측 레일과 상단 요약을 고정한다.

**Color Philosophy**: 간판 레드, 학교 앞 분식집 노랑, 검정 메뉴판 잉크, 스테인리스 회색을 사용한다. 레드는 브랜드 에너지를, 노랑은 실행 가능성을, 회색은 주방 운영의 현실성을 의미한다.

**Layout Paradigm**: 화면을 커다란 메뉴 보드처럼 나누고, 비용·매출·손익을 각각 다른 메뉴군으로 배치한다. 일반적인 카드 그리드 대신 사선 가격표, 가로 레일, 라벨형 섹션을 사용한다.

**Signature Elements**: 네모난 메뉴 라벨, 가격표 점선, 키오스크 버튼, 주방 오더 티켓, 스테인리스 질감의 구분선을 사용한다.

**Interaction Philosophy**: 입력은 키오스크 버튼을 누르는 느낌으로 빠르고 명확해야 하며, 결과는 매장 운영판에 숫자가 갱신되는 것처럼 즉각적이어야 한다.

**Animation**: 클릭 시 버튼이 2px 눌리는 물리적 피드백을 주고, 섹션 전환은 오더 티켓이 옆으로 밀려 들어오는 방식으로 처리한다. 과도한 스프링 애니메이션은 피한다.

**Typography System**: 제목은 굵은 고딕 디스플레이, 본문은 Noto Sans KR, 숫자는 IBM Plex Sans KR 또는 Pretendard numeric 스타일을 사용한다. 가격은 굵게, 설명은 좁은 행간으로 메뉴판의 밀도를 살린다.

**Design Movement**: SNS 콜라보 캠페인 랩. 삼첩분식의 월간 콘텐츠, 시즌 박스, 캐릭터 협업, 신메뉴 확산 구조를 반영해 “브랜드가 매달 움직이는 실험실”처럼 표현한다.

**Core Principles**: 첫째, 매출 시뮬레이션은 단순 평균값이 아니라 프로모션과 신메뉴 이슈가 반영되는 성장 시나리오로 보여야 한다. 둘째, 화면은 SNS 카드, 캠페인 캘린더, 주문 퍼널이 연결된 구조여야 한다. 셋째, 창업 상담 사용자는 마케팅이 매출에 미치는 가정을 쉽게 조정할 수 있어야 한다. 넷째, 시각 요소는 젊고 활발하되 결과표는 보수적으로 읽혀야 한다.

**Color Philosophy**: 레드와 옐로우를 기본으로 하되, 캠페인 스티커용 민트·핑크·라벤더를 제한적으로 사용한다. 다만 과도한 파스텔은 재무 도구의 신뢰성을 희석하므로 보조 배지에만 배치한다.

**Layout Paradigm**: 타임라인형 비대칭 레이아웃을 사용한다. 상단은 캠페인 요약, 중간은 입력 조정, 하단은 월별 손익 변화 그래프로 구성해 “콘텐츠 → 검색 → 주문 → 매출” 퍼널을 화면 구조로 만든다.

**Signature Elements**: SNS 포스트 프레임, 저장/공유 아이콘을 닮은 지표 배지, 캠페인 캘린더, 신메뉴 태그, 그래프 위의 스티커 핀을 사용한다.

**Interaction Philosophy**: 사용자가 프로모션 강도나 객단가를 바꾸면 그래프 위 캠페인 핀이 이동해 마케팅 가정이 수익 구조에 어떤 영향을 주는지 보여준다.

**Animation**: 타임라인 핀은 부드럽게 이동하고, 캠페인 카드가 등장할 때는 소셜 피드가 로딩되는 듯한 짧은 stagger를 사용한다. 결과 수치에는 과도한 축하 애니메이션을 넣지 않는다.

**Typography System**: 제목은 둥글고 친근한 한글 디스플레이, 본문은 Pretendard, 캡션은 좁은 자간의 Noto Sans KR을 사용한다. SNS 카드 제목과 재무표 제목의 무게 차이를 명확히 둔다.

## 최종 선택

최종 디자인 방향은 **프리미엄 분식 패키지 콜라주**로 선택한다. 이 접근은 삼첩분식의 공식 브랜드 요소인 레드·옐로우 컬러, 삼첩 구조, 프리미엄 패키지, 시즌 박스와 가장 직접적으로 연결되며, 창업 상담용 수익 계산기의 신뢰성과 브랜드의 경쾌함을 동시에 표현하기에 적합하다.
