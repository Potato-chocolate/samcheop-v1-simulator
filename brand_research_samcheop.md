# 삼첩분식 브랜드 조사 및 웹 구현 방향

작성자: **Manus AI**

## 조사 목적

본 문서는 삼첩분식 가맹 상담용 웹 시뮬레이터를 삼첩분식 브랜드에 맞게 재정의하기 위한 근거 자료이다. 조사 범위는 공식 홈페이지, 공식 앱 소개 페이지, 공식 홈페이지 내 브랜드 스토리와 메뉴·뉴스 영역을 중심으로 구성했으며, 웹 구현에 반영할 **브랜드 정체성, 메뉴 톤, 색감, 고객 경험 키워드**를 도출했다.

> 삼첩분식 공식 브랜드 페이지는 브랜드 문구를 “떡볶이의 새로운 기준, 삼첩분식”으로 제시하고, 매운맛 중심의 떡볶이 시장에 다양성과 취향별 선택 경험을 더한다고 설명한다.[1]

## 핵심 브랜드 정체성

삼첩분식은 떡볶이를 중심으로 하지만 단일한 ‘매운 분식’ 이미지에 머물지 않는다. 공식 브랜드 설명은 **매콤함과 달콤함, 취향 선택, 다양한 맛 연구**를 전면에 내세우며, “spicy! sweet! samcheop!”이라는 표현으로 브랜드의 감각적 톤을 압축한다.[1] 또한 브랜드의 핵심 가치는 맛의 새로운 기준, 정직과 신뢰, 특별한 가치 경험, 지속 가능성으로 제시되어 있어, 단순한 원가 계산기보다는 **상담자가 신뢰 가능한 근거와 브랜드 경험을 함께 보여주는 시뮬레이터**로 설계하는 것이 적합하다.[1]

| 구분 | 조사 근거 | 웹 구현 시사점 |
|---|---|---|
| 브랜드 슬로건 | “떡볶이의 새로운 기준”[1] | 히어로 카피는 계산 도구보다 ‘가맹 상담 기준표’ 성격을 강조한다. |
| 맛의 방향 | 매콤함·달콤함·취향별 선택[1] | 입력 단계를 1첩·2첩·3첩 구조로 나누어 선택 경험을 만든다. |
| 고객 경험 | 협업 이벤트와 신선한 아이디어[1] | 패키지, 콜라보, 이벤트 감성을 시각 요소로 반영한다. |
| 신뢰 요소 | 정직과 신뢰, 정도 경영[1] | 계산 가정과 단위, 면제 항목, BEP를 숨기지 않고 노출한다. |
| 지속 가능성 | 재사용 가능한 종이 패키지 언급[1] | 종이 패키지·영수증·박스 질감을 UI 모티프로 활용한다. |

## 메뉴 톤과 콘텐츠 구조

공식 홈페이지 메뉴는 신메뉴, 1첩, 2첩, 3첩, 더하다, Set으로 분류되어 있다.[2] 대표 메뉴는 삼첩 떡볶이, 바질크림 떡볶이, 로제 떡볶이, 마라로제 떡볶이, 양배추 떡볶이 등으로 구성되어 있으며, 사이드에는 모둠튀김, 순살치킨, 대구막창, 주먹밥, 순대튀김 등이 포함된다.[2] 이 구조는 상담 웹에서 매출·비용·개설비용을 설명할 때 **1첩·2첩·3첩이라는 단계형 은유**로 전환하기 좋다.

삼첩분식의 신메뉴와 뉴스 흐름은 로제샹궈 떡볶이, 저당 양배추볶이, 마라로제 출시 5주년 이벤트, 오트사이드 콜라보, 쿠키런·짱구·틴틴팅클 콜라보 등으로 이어진다.[2] 따라서 웹의 시각 톤은 전통적인 분식점의 소박함보다 **팝업 이벤트, 시즌 패키지, 캐릭터 협업에 맞는 경쾌한 그래픽 톤**이 적합하다.

| 메뉴·콘텐츠 축 | 공식 자료에서 확인한 요소 | 상담 UI 반영 방향 |
|---|---|---|
| 1첩·2첩·3첩 | 공식 메뉴 분류와 세트 구성[2] | 매출 입력, 손익 계산, 개설비용 산출을 3단계로 구성한다. |
| 로제·마라·바질 | 색다른 맛의 확장[1] [2] | 붉은색 하나가 아니라 크림·옐로우·그린 보조색을 제한적으로 사용한다. |
| 삼첩박스 | 2020년 이후 반복되는 시즌 패키지 뉴스[2] | 카드·박스·영수증형 패널을 사용해 브랜드 기억을 강화한다. |
| 앱 주문·멤버십 | 배달·포장, 인기 메뉴, 멤버십 혜택 제공[3] | 저장 리포트와 공유 링크를 ‘상담 후 다시 확인하는 멤버십형 경험’으로 설명한다. |

## 색감 및 비주얼 방향

공식 브랜드 페이지는 메인 컬러를 떡볶이를 상징하는 **레드 컬러**와 경쾌한 **옐로우 포인트 컬러**로 설명한다.[1] 심볼 로고의 세 가지 색상은 떡볶이, 토핑, 사이드를 함께 즐기는 즐거움을 표현한다고 되어 있다.[1] 이 때문에 기존 v2 단막의 브라운·고깃집 중심 디자인은 삼첩분식에는 맞지 않으며, 삼첩분식 웹은 **칠리 레드, 패키지 옐로우, 크림 베이지, 잉크 블랙** 중심으로 재정의하는 편이 일관적이다.

| 디자인 토큰 | 권장 방향 | 근거 및 역할 |
|---|---|---|
| Primary Red | 떡볶이 소스와 브랜드 메인 컬러 | 공식 페이지가 레드를 떡볶이 상징 컬러로 설명한다.[1] |
| Accent Yellow | 패키지·버튼·강조 배지 | 공식 페이지가 옐로우를 경쾌한 포인트 컬러로 설명한다.[1] |
| Warm Cream | 영수증·상담지 배경 | 빨강·노랑 대비를 완화하고 숫자 가독성을 확보한다. |
| Ink Black | 금액·계산 결과·주의 문구 | 상담 도구의 신뢰성과 정보 밀도를 유지한다. |
| Paper Texture | 삼첩박스·종이 패키지 연상 | 공식 페이지의 지속 가능성·종이 패키지 메시지와 연결된다.[1] |

## 고객 경험 키워드

삼첩분식 앱 소개는 사용자가 주변 매장을 선택해 취향대로 주문하고, 포장부터 배달까지 이용할 수 있으며, 인기 메뉴 탐색과 멤버십 혜택을 제공한다고 설명한다.[3] 이는 삼첩분식의 고객 접점이 단순 매장 방문을 넘어 **앱 기반 주문, 배달·포장 편의, 프로모션, 멤버십**으로 확장되어 있음을 보여준다. 상담용 시뮬레이터에서도 ‘한 번 계산하고 끝나는 화면’이 아니라 저장, 공유, PDF 출력, 공개 리포트 확인 기능을 통해 상담 이후 재방문 가능한 경험을 제공하는 것이 브랜드의 디지털 접점과 잘 맞는다.

| 고객 경험 키워드 | 의미 | 현재 웹 적용 상태 |
|---|---|---|
| 취향 선택 | 메뉴와 맛을 고객이 고르는 구조 | 매출·평수·운영형·간판·인력 조건을 직접 조정하도록 구성했다. |
| 즉시성 | 앱 주문과 상담 현장 계산 모두 빠른 피드백이 중요 | 입력값 변경 시 손익·BEP·개설비용이 즉시 갱신된다. |
| 재확인 | 상담 후 리포트를 다시 확인해야 함 | DB 저장 리포트, URL 공유, PDF 출력, 공개 상세 페이지를 구현했다. |
| 신뢰 | 비용 항목과 프로모션 차감의 명확한 설명 | 본사비용 정상가와 면제 차감액을 분리 표시했다. |
| 경쾌함 | 콜라보·시즌 패키지 중심의 브랜드 활동 | 패키지형 카드, 영수증형 리포트, 레드·옐로우 강조를 적용했다. |

## v2 단막 디자인에서 삼첩분식으로의 재정의

기존 v2 단막 방향이 브라운·고깃집·구이집 정서에 가까웠다면, 삼첩분식 전용 웹은 **배달 분식, 패키지, 떡볶이 소스, 시즌 콜라보**의 감각으로 전환되어야 한다. 따라서 화면은 어두운 숯불·목재 질감보다 밝은 크림 배경, 종이 질감, 빨강·노랑 대비, 영수증형 계산 리포트를 중심으로 구성하는 것이 적절하다.

| 기존 v2 단막 방향 | 삼첩분식 전용 재정의 |
|---|---|
| 브라운·고기·매장 식사 중심 | 레드·옐로우·떡볶이·패키지 중심 |
| 고깃집 메뉴판형 정보 구조 | 1첩·2첩·3첩 단계형 상담 구조 |
| 매장 인테리어 강조 | 배달·포장·홀 겸업 상담 조건 강조 |
| 정적인 견적표 | 입력값에 따라 즉시 변하는 손익·BEP 리포트 |
| 단일 화면 공유 | URL 복원, 서버 저장, 공개 리포트, PDF 출력으로 상담 후속 경험 확장 |

## 구현 기준 요약

삼첩분식 상담용 웹은 브랜드 조사 결과에 따라 **프리미엄 분식 패키지 콜라주** 방향으로 유지하는 것이 타당하다. 화면의 핵심은 브랜드의 재미를 보여주는 것이 아니라, 창업 상담에서 매출·비용·개설비용을 신뢰 있게 설명하는 데 있다. 그러므로 감각적 장식은 레드·옐로우·종이 패키지·영수증 패널로 제한하고, 계산 결과와 가정은 표와 숫자 중심으로 명확하게 노출해야 한다.

> 결론적으로, 삼첩분식 시뮬레이터는 “예쁜 브랜드 소개 페이지”가 아니라 **브랜드 톤을 입힌 상담 계산 리포트 도구**로 정의한다. 이 방향은 공식 브랜드가 말하는 맛의 다양성, 정직과 신뢰, 특별한 경험, 앱 기반 편의성과 일관된다.[1] [3]
